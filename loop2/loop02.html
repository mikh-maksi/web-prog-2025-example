Цикли

Класифікація за умовами закінчення:

По кількості разів виконання (параметр)

for(let i=0;i<=10;i+=1){
console.log(i);
}

За кількістю отриманих знанчень

let n=1; for(let i=0;n<=10;i+=1){ if (i%2==0){ n+=1; console.log(i); } }

За за значенням певної змінної

let sum = 0; for(let i=0;sum<=100;i+=1){ if (i%2==0){ sum+=i; console.log(i); console.log("sum = "+sum); } }

За умовами роботи

Цикл з параметром

for(let i=0;i<=10;i+=1){
console.log(i);
}
